# on-cns-archive

## 简介

`on-cns-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 页面的集中存储与展示空间。

## 目录说明

```
├── pages/          # 存放所有归档的 HTML 页面
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── index.html      # 仓库根目录入口页面
└── README.md       # 本文件
```

- `pages/`：每个归档页面以独立文件或子目录形式存放，命名清晰，便于检索。
- `assets/`：页面所需的公共或私有资源文件。
- `index.html`：根目录下的入口页面，可用于导航或展示归档列表。

## 页面归档说明

- 本仓库中的 HTML 页面均为独立归档，彼此无依赖关系。
- 页面可能包含内联样式与脚本，也可能引用 `assets/` 中的外部资源。
- 归档内容不涉及具体网站或域名，仅为通用 HTML 页面集合。
- 页面访问方式：通过 GitHub Pages 或直接打开文件均可正常浏览。

## 维护说明

- 欢迎提交包含独立 HTML 页面的 Pull Request，请将页面放入 `pages/` 目录。
- 请确保页面不含外部资源链接（除本仓库 `assets/` 外），避免引用失效。
- 建议为每个页面添加简要注释或说明，便于他人理解归档目的。
- 本仓库不提供内容审核或筛选，但请遵守开源社区基本规范。
